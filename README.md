# Vape Shops in Bataan API (Frontend Demo)

A frontend demo for the Bataan Vape Shops API built with **Vite**, **React**, **Tailwind CSS**, **shadcn/ui**, and **mapcn**.

## Installation

```bash
# clone repository
git clone https://github.com/boontazabooraw/veepshtest.git

# go into project
cd veepshtest

# install dependencies
npm install
```

## Development

```bash
npm run dev
```

The app will start at:

```
http://localhost:5173
```

## Build

```bash
npm run build
```
